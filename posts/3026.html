min.css 和.css

两者仅仅是文件大小、有无注释、格式美化（可读性）上的区别，使用上（作用效果）并没有什么区别。压缩和未压缩的区别，.min 是压缩版的（去掉多余的注释、空格等）文件较小，易于加载，另外的就是未压缩的了文件稍大，保留完整的注释、可读性强的标准格式化文件。一般生产环境用.min.css（加载速度快，提升体验） , 开发试验阶段用.css（便于理解）。

参考自 min.css 和普通 css 有什么不同 – 范仁义 – 博客园 (cnblogs.com)

关于修改样式

直接在主题编辑器中做出的的修改是无法在更新时保留的，可以在主题 – 自定义 – 额外 CSS 进行你想要的修改，只需对你想要修改的某个元素进行修改就行了，在这里修改的样式在更新时是可以保留的。而且通过内置的 css 编辑器是可以进行效果实时预览的。

纯文本粘贴

众所周知粘贴是 Ctrl + V，但是这种粘贴是带格式的。如果想要清除格式，也就是纯文本粘贴，只需按下 Ctrl + Shift + V 。

测试域名解析是否生效

Windows 命令窗口中输入 nslookup 域名，例如

nslookup crowya.com
